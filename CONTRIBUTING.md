# Contributing to SkillMesh

Thank you for your interest in SkillMesh! We welcome contributions from the community to help make this decentralized P2P professional network even better.

## How to Contribute

### 1. Reporting Bugs
- Use the **Bug Report** issue template.
- Provide a clear description and steps to reproduce.

### 2. Suggesting Features
- Use the **Feature Request** issue template.
- Explain the "Why" and "How" of your proposal.

### 3. Submitting Pull Requests
- Fork the repository.
- Create a feature branch (`git checkout -b feature/amazing-feature`).
- Ensure your code follows the existing style.
- Submit a PR with a clear description of the changes.

## Development Setup

1. Clone the repo: `git clone https://github.com/your-username/SkillMesh-P2P.git`
2. Setup the Professional Node:
   - `cd professional-node`
   - `npm install`
   - `node node.js`
3. Test the User Client:
   - Open `client-app/index.html` in your browser.
